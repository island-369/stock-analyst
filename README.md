## 中文

股票多维度分析 Skill，支持 **港股 / A股 / 美股**，整合多源数据进行技术面、基本面、资金面和宏观面综合分析，自动生成图文结合的结构化分析报告。

### 功能

- 🔍 **多市场支持** — 港股（如 0700）、A股（如 600519）、美股（如 AAPL）
- 📊 **技术分析** — K 线图、分时图、均线、MACD、RSI、布林带等
- 📰 **多源数据** — 东方财富、yfinance、akshare 联合采集
- 📝 **自动报告** — Markdown 报告 + 专业 CSS 排版 HTML + PDF 导出
- ⚡ **一键环境** — `setup.sh` 自动创建 Python venv 并安装依赖

### 快速开始

```bash
# 1. 初始化环境（首次运行，只需执行一次）
bash scripts/setup.sh

# 2. 采集数据并生成图表
source scripts/venv/bin/activate
python scripts/fetch_all.py --code 0700 --market hk

# 3. 报告由 AI 模型根据模板自动撰写
#    参考 references/report_template.md
```

### 目录结构

```
├── SKILL.md                  # Skill 定义与流程指令
├── package.json
├── scripts/
│   ├── setup.sh              # 一键环境初始化
│   ├── fetch_all.py          # 数据采集 + 图表生成
│   ├── fetch_ir.py           # 投资者关系数据采集
│   ├── text_chart.py         # ASCII 文本图表（终端展示用）
│   ├── md2html.py            # Markdown → 专业 HTML
│   ├── export_report.py      # Markdown → PDF
│   └── requirements.txt
└── references/
    ├── report_template.md    # 报告模板与写作规则
    ├── hk_stock_knowledge.md # 港股相关知识
    └── sensitive_companies.md
```

### 环境要求

- Python ≥ 3.10
- 多模态 AI 模型支持（如 glm-5v-turbo，需能读取图片）

## English <a id="english"></a>

A multi-dimensional stock analysis skill supporting **HK / A-Share / US** markets, integrating data from multiple sources for technical, fundamental, capital-flow, and macro analysis, generating structured reports with visualizations automatically.

### Features

- 🔍 **Multi-market** — HKEX (e.g. 0700), A-Shares (e.g. 600519), US (e.g. AAPL)
- 📊 **Technical Analysis** — K-line charts, intraday charts, MA, MACD, RSI, Bollinger Bands
- 📰 **Multi-source Data** — Eastmoney, yfinance, akshare
- 📝 **Auto Reports** — Markdown + styled HTML + PDF export
- ⚡ **One-command Setup** — `setup.sh` bootstraps Python venv with all dependencies

### Quick Start

```bash
# 1. Setup environment (first run only)
bash scripts/setup.sh

# 2. Fetch data & generate charts
source scripts/venv/bin/activate
python scripts/fetch_all.py --code AAPL --market us

# 3. Reports are auto-generated by AI using the template at
#    references/report_template.md
```

### Requirements

- Python ≥ 3.10
- Multimodal AI model (e.g. glm-5v-turbo) capable of reading images
