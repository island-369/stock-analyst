# 港股分析专业知识

## 港股特有的驱动力

### 南向资金（北水）
- 内地资金通过港股通买卖港股的资金流
- 对中大型港股（尤其是科技股）有显著定价权
- 连续多日大幅净买入通常是短期看涨信号
- 关注指标：连续方向天数、日均净买入量、持股比例变化

### ADR 隔夜效应
- 有 ADR 的港股（腾讯TCEHY、阿里BABA等），ADR 收盘价暗示次日港股开盘方向
- ADR 溢价 >1% → 次日大概率高开
- ADR 折价 >1% → 次日大概率低开
- 传导机制：美股时段发生的新消息先在 ADR 定价，次日港股开盘跟随

### 联系汇率制
- 港元盯住美元（7.75-7.85区间）
- 美联储加息 → 港元利率跟随 → 流动性收紧 → 利空港股
- USD/HKD 接近 7.85（弱方兑换保证）时最危险

### 做空机制
- 港股允许裸卖空
- 做空机构报告（Muddy Waters, Citron, Hindenburg）对股价有立竿见影的影响
- 卖空股数骤增 = 空头建仓；卖空骤降 = 空头平仓（可能是短期底部信号）

## 事件分类及典型影响

| 事件类型 | 典型影响方向 | 影响时效 | 说明 |
|---------|-------------|---------|------|
| 财报超预期 | 强利好 | 1-5天 | 开盘跳空后可能持续 |
| 财报不及预期 | 强利空 | 1-5天 | 跌幅往往大于涨幅（不对称） |
| 大规模回购 | 利好 | 中期 | 信号意义>实际影响，显示管理层信心 |
| 分析师集体上调 | 中性偏好 | 1-3天 | 评级变动的影响不如目标价调整 |
| 中国行业监管 | 强利空 | 中期 | 如2021年反垄断，可持续压制估值 |
| 美国制裁/审查 | 利空 | 长期不确定 | 悬而未决时持续压制风险偏好 |
| 纳入/剔出指数 | 利好/利空 | 1-2周 | 被动资金调仓带来确定性买卖盘 |
| 大股东减持 | 利空 | 短期 | 前大股东（如腾讯的Naspers/Prosus）减持影响已减弱 |
| 配股增发 | 短期利空 | 1-3天 | 摊薄效应，但如果募资用途好则影响有限 |
| 管理层变动 | 不确定 | 中长期 | 需看具体人选和市场解读 |

## 估值参考区间（分行业）

**⚠️ 重要：不要用"PE<15便宜/15-25合理/>25偏贵"的通用框架判断所有股票。不同行业差异巨大。**

| 行业类别 | PE 合理区间 | PB 合理区间 | 说明 |
|---------|------------|------------|------|
| 银行/金融 | 5-10x | 0.5-1.5x | 高杠杆、低增长，PE 低是正常的 |
| 消费（食品饮料/零售） | 20-35x | 3-8x | 现金流稳定，可给溢价 |
| 科技（互联网/软件） | 25-50x | 3-10x | 高增长支撑高 PE，关注 PS/FCF |
| 医药（创新药/器械） | 30-60x | 3-8x | 研发投入大，PE 偏高；仿制药 PE 更低 |
| 周期（钢铁/煤炭/化工） | 8-15x | 1-3x | 盈利波动大，PE 低未必便宜（周期顶部） |
| 公用事业（电力/水务） | 15-25x | 1-3x | 稳定分红，看股息率更准 |
| 地产 | 5-10x | 0.5-1.5x | 看 NAV 折价而非 PE |
| 半导体/芯片 | 30-80x | 3-10x | 周期+成长，PE 波动大，看 PS 和产能 |
| 新能源（光伏/锂电） | 20-40x | 2-5x | 高增长期 PE 偏高，关注产能过剩风险 |

**判断原则：**
1. **优先用 `peer_comparison.industry_avg` 的行业均值**做对比，而非硬编码阈值
2. 高增长（营收增速>30%）公司 PE 可上浮 50-100%
3. 周期股 PE 低可能是周期顶部（盈利高点），PE 高可能是周期底部（盈利低点）—— 反向看
4. 亏损股 PE 无意义，改看 PS、PB、FCF Yield

| 指标 | 通用参考（仅兜底，优先用行业均值） |
|------|------|
| FCF Yield | >6% 便宜 / 3-6% 合理 / <3% 偏贵 |
| 股息率 | >5% 高 / 2-5% 合理 / <2% 低 |
| 资产负债率 | <40% 健康 / 40-70% 正常 / >70% 偏高（金融地产除外） |

## A 股特有机制

### 北向资金（陆股通）

- 境外资金通过沪深港通买卖 A 股的资金流
- 又称"聪明钱"——北向资金对 A 股蓝筹和消费龙头有定价影响力
- 连续多日大幅净买入通常是中短期看涨信号
- 关注指标：每日净买入额（亿元）、连续方向天数、对个股的净买入额

与港股的南向资金对称：
- **南向资金**：内地资金买港股
- **北向资金**：境外资金买 A 股
- 分析 A 股时看北向，分析港股时看南向

### 涨跌停制度

- 普通 A 股：±10% 涨跌停
- 创业板/科创板：±20% 涨跌停
- ST 股票：±5% 涨跌停
- 港股和美股没有涨跌停制度

涨跌停对分析的影响：
- 封涨停 = 极度看多，买盘远大于卖盘，但无法判断真实均衡价格
- 封跌停 = 极度看空，恐慌出逃，流动性枯竭
- 连续涨停/跌停后，开板当天成交量暴增是关键观察点

### 融资融券

- A 股的杠杆交易数据（两融余额/融资买入额）
- 融资余额持续增加 → 市场加杠杆做多
- 融资余额拐头减少 → 去杠杆，偏空信号
- 融券卖出（做空）在 A 股规模很小，影响有限

### A 股特有的宏观因子

| 因子 | 怎么看 | 传导路径 |
|------|--------|---------|
| LPR（贷款市场报价利率） | 降 LPR → 利好消费/地产 | PBOC→LPR→企业融资成本→盈利预期 |
| 社融数据 | 超预期 → 实体经济在加杠杆 | 信贷投放→企业扩张→盈利增长 |
| PMI | >50扩张，<50收缩 | 制造业景气→工业股/原材料 |
| 国常会/政治局会议 | 关注政策表述变化 | 政策定调→板块轮动 |
| 印花税/IPO节奏 | 降印花税=重大利好 | 交易成本→市场情绪→成交量 |
